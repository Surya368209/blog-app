**College Blogging Application**

A full-stack College Blogging Platform where students and teachers can create, view, and interact with posts securely.
This project is built using **Spring Boot** for the backend and **React** for the frontend, following real-world architecture and best practices.


### Tech Stack

### Backend (security)
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate / JPA
- MySQL
- Maven

### Frontend (college-blog-frontend)
- React (Vite)
- JavaScript (ES6+)
- HTML5 / CSS3
- Fetch API
- Responsive UI

## Features
- User Authentication using JWT
- Role-based access (Student / Teacher / Admin)
- Create, view, and manage blog posts
- Secure REST APIs
- Separation of frontend and backend
- Clean, scalable folder structure
